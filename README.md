# Search Artist App

Состоит из:
экрана приветствия из 3х фрагментов
активити с автокомплитным полем для ввода имени артиста
активити с webView для открытия ссылки, полученной через Api

Использованные библиотеки:
- Dagger 2
- RxJava
- Retrofit
